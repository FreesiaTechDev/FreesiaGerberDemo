# Freesia Gerber Demo

## 目的 / Purpose

此 repository 提供一個 WPF demo application，用來展示如何使用 `FreesiaGerberLib.dll`。

Demo source code 可作為學習、整合參考與商業類別庫評估用途。

This repository provides a WPF demo application that demonstrates how to use `FreesiaGerberLib.dll`.

The demo source code is provided for learning, integration reference, and evaluation of the commercial library workflow.

## 系統需求 / Requirements

- Visual Studio with WPF desktop development support
- Demo app 目標 framework 為 .NET Framework 4.6.2
- `FreesiaGerberLib.dll` target framework 為 `netstandard2.0`
- 使用 `FreesiaGerberLib.dll` 需要有效的 hardware dongle

- Visual Studio with WPF desktop development support
- The demo app targets .NET Framework 4.6.2
- `FreesiaGerberLib.dll` targets `netstandard2.0`
- Using `FreesiaGerberLib.dll` requires a valid hardware dongle
